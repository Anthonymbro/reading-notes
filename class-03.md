# Class-03

### What is Version Control? 

Version control is software that records our changes. It allows you to revisit several versions of files and records changes.

### What is cloning in Git?

Cloning is when you clone the URL of a GitHub page to VS Code from what I know.

### What is the command to track and stage files?

Git add 

### What is the command to take a snapshot of your changed files?

Git commit -m “reason why you are doing it”

### What is the command to send your changed files to Github?

Git push origin master
